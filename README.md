# Solo MMORPG Prototype

An original solo-developed MMORPG prototype inspired by the design values of classic WoW-era MMORPGs and modern solo-indie development efforts like Elegon.

Core direction:
- open world feeling
- meaningful progression
- readable low-poly 3D world
- always-online multiplayer architecture
- no pay-to-win systems
- strong early vertical slice before content scaling

Reference inspiration used for planning:
- Elegon site vision: open-world MMO, earned progression, social world, long-term live development
- Elegon devlog 40 title/summary: maps, spell queueing, optimization
- roadmap signals: quests, nameplates, emotes, compass, stat clarity, movement polish, water/swimming

Important constraint:
This project should be original in code, assets, naming, lore, UI, and worldbuilding. We can borrow structure and product lessons, not copy proprietary content.
